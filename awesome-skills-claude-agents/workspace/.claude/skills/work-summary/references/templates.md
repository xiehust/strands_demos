# Work Summary Report Templates

- [Task 3 with brief description]

- [Blocker 1 and impact]
- [Challenge 1 and how addressing it]

## Next Steps
- [Priority 1 for next period]
- [Priority 2 for next period]
- [Priority 3 for next period]

- [Metric 1]: [Value]

## Template Customization Guidelines

### Adapting Templates
1. **Remove unused sections**: If a section doesn't apply, remove it
2. **Adjust depth**: Scale detail level based on audience and purpose
3. **Customize headers**: Adapt section titles to match organizational terminology
4. **Add sections**: Include organization-specific sections as needed
5. **Format for audience**: More visual for executives, more detail for technical audiences

### Format-Specific Adaptations

#### DOCX Reports
- Use consistent heading hierarchy (Heading 1, 2, 3)
- Include table of contents for reports over 5 pages
- Use tables for structured data
- Add page numbers and headers/footers
- Use bullet points for lists

#### PPTX Reports
- One main idea per slide
- Use visual hierarchy (large headings, concise bullets)
- Limit text (6-8 bullets max per slide)
- Use charts and graphs for data
- Include section divider slides
- Keep slides visually consistent

#### Markdown Reports
- Use standard markdown formatting
- Include links to relevant resources
- Use code blocks for technical content
- Keep tables simple and readable
- Use emoji sparingly for status indicators (‚úÖ üü° ‚ùå)
