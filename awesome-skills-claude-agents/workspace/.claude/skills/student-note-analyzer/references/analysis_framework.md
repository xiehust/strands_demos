# Note Analysis Framework

## Knowledge Point Identification Criteria

### Primary Knowledge Points
- **Definitions**: Core concepts and terminology
- **Principles**: Fundamental rules or theories
- **Processes**: Step-by-step procedures or methods
- **Relationships**: Connections between concepts
- **Applications**: Practical uses or examples

### Secondary Knowledge Points
- **Supporting details**: Evidence, statistics, examples
- **Context**: Historical background or situational information
- **Comparisons**: Similarities and differences
- **Exceptions**: Special cases or limitations

## Understanding Level Assessment

### Level 1: Recognition (Basic)
**Indicators in notes:**
- Lists definitions without explanation
- Copies verbatim from source material
- Minimal personal interpretation
- No examples or applications

**Assessment**: Surface-level understanding

### Level 2: Comprehension (Developing)
**Indicators in notes:**
- Restates concepts in own words
- Provides basic examples
- Shows understanding of main ideas
- Makes simple connections

**Assessment**: Adequate understanding

### Level 3: Application (Proficient)
**Indicators in notes:**
- Applies concepts to new scenarios
- Creates original examples
- Identifies patterns and relationships
- Asks meaningful questions

**Assessment**: Strong understanding

### Level 4: Analysis (Advanced)
**Indicators in notes:**
- Compares and contrasts multiple concepts
- Evaluates strengths and limitations
- Synthesizes information from multiple sources
- Draws independent conclusions

**Assessment**: Deep understanding

## Note Quality Indicators

### Well-Structured Notes Include:
- Clear hierarchy (headings, subheadings)
- Logical flow and organization
- Visual elements (diagrams, charts, bullet points)
- Key terms highlighted or emphasized
- Personal annotations or questions
- Cross-references between topics

### Common Note Deficiencies:
- Missing key concepts from curriculum
- Lack of structure or organization
- Incomplete explanations
- No examples or applications
- Absent connections between topics
- Missing context or background
